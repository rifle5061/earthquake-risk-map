# 地震危険度マップ v1

日本地図にプレート・海溝ライン、地震発生地点、期間切替、危険度カードを表示するMVPです。

## 内容

- `index.html`：画面本体
- `style.css`：デザイン
- `app.js`：地図・期間切替・サンプル地震データ・リスク計算

## 現在の仕様

- 地震データはサンプルデータです。
- プレート/海溝ラインは概略です。
- リスクスコアはβ版の仮計算です。
- 地震の発生日・場所・規模を予測するものではありません。
